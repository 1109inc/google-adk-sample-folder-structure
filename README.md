# FastAPI Agent Backend Starter

A simple FastAPI backend starter for agentic projects.

This project includes user authentication, database setup, API routing, and an AI agent workflow using ADK. It is meant to be a clean base that can be reused for future backend + agent projects.

## Features

- FastAPI backend structure
- JWT-based authentication
- SQLAlchemy database setup
- Modular API routing
- Reusable dependency setup
- ADK-based chat agent integration
- Beginner-friendly comments across the codebase
